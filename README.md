# 🧠 Multi-Theme Branching Quiz

An interactive quiz with two distinct paths: Space Exploration or Stranger Things universe.

## 🚀 Key Features
* **Dual Paths**: Choose your topic at the start.
* **Dynamic Endings**: Your score and choices lead to different final results.
* **Logic**: Implemented complex branching based on user input.

*Note: The content is currently in Russian.*
